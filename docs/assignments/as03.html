Creating a Function: regplot

This function combines lm(), plot(), and abline() to create a regression fit plot.

regplot <- function(x, y) {
  fit <- lm(y ~ x)
  plot(x, y)
  abline(fit, col = "red")
}

What Is the Problem?

table(TEDS_2016$votetsai, useNA = "ifany")

   0    1 <NA> 
 471  790  429

The problem is that votetsai is a categorical dependent variable with only two categories, usually coded as 0 and 1. A regular linear regression model assumes that the dependent variable is continuous, but vote choice is binary.

How Can Prediction Be Improved?

Prediction can be improved by using a model designed for a categorical dependent variable. For a two-category outcome like votetsai, logistic regression is more appropriate than linear regression because it predicts probabilities between 0 and 1.

model_logit <- glm(
  votetsai ~ age + edu + income,
  data = TEDS_2016,
  family = binomial
)
